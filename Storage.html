Storage


Computer data storage, often called storage or memory, is a technology
made up of computer components and recording media that are used to
keep data. It is a core function and fundamental component of
computers. Data can be stored in hard drives, ssd's (solid state drives) and etc.
Data can be written on and read off of these drives.
