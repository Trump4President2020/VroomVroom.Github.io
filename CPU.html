What Does CPU Stand For?

CPU stands for Central Processing Unit.

What Does the CPU Do?

A central processing unit (CPU) is the electronic circuitry within a computer that carries out the instructions
of a computer program by performing the basic arithmetic, logical, control and input/output operations specified by
the instructions.The term "central processing unit" has been used at least since the early 1960s. Traditionally, the
term "CPU" refers to a processor, more specifically its processing and control unit, discriminate these core elements
of a computer from external components such as main memory and input/output circuitry.
