KEYBOARD

A keyboard is a 'typewriter-like' deivice, which is used as a text entry interface to enter text into a text editor, word processor or other programs. Symbols on keyboards usually correspond to one or more characters, which can be accessed through a combination of some, such as ctrl or alt. The three main keyboard layouts in Europe are AZERTY in France and Belgium, QWERTZ in most German speaking and Central European countries (some Slavic countries as well) and QWERTY used pretty much everywhere else. Keyboards are currently the main input devices for computers
